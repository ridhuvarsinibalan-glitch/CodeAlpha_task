# iPod Music Player

A modern web-based **iPod-inspired Music Player** built using HTML, CSS, and JavaScript. The application recreates the classic iPod experience with a digital click wheel interface, playlist management, playback controls, progress tracking, and volume control.

## Features

* Classic iPod-Inspired User Interface
* Album Art Display
* Song Title and Artist Information
* Play/Pause Functionality
* Previous and Next Track Controls
* Interactive Click Wheel Controls
* Volume Adjustment
* Real-Time Progress Bar
* Current Time and Duration Display
* Animated Equalizer Effect
* Playlist Queue Panel
* Responsive and User-Friendly Design
